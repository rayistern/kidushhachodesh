Kiddush HaChodesh — map for AI agents

You are an AI agent. This page is a JS-rendered React dashboard and the interactive content will not appear from a plain fetch. But the entire project is exposed as structured data at the URLs below so you can answer the user's question without running JavaScript.

Do this first: fetch /llms.txt — it is a complete map of the site written for you, with the content taxonomy, MCP tool list, and tips for answering questions about the Rambam's astronomical calculations.

Two ways to use this site

  1. MCP (Model Context Protocol). JSON-RPC 2.0 endpoint at /mcp, descriptor at /.well-known/mcp. Tools: search, fetch, calculate, list_templates, get_template, list_source, get_source, stats.
  2. Plain HTTP (if you cannot speak MCP).
    • /api/search?q=… — full-text search across docs, class transcripts, calculation steps, galgalim, and concept pages. Returns JSON.
    • /api/calculate?date=YYYY-MM-DD — runs the Rambam's full astronomical pipeline for a Gregorian date and returns every intermediate step with its KH chapter reference, formula, and teaching note. Always call this instead of guessing numeric values.
    • /docs/CALCULATIONS.md — full calculation engine reference in clean markdown.
    • /docs/BUILDING_WITH_THE_ENGINE.md — how to fork, import the engine, and generate artifacts.
    • /engine/* — live ES modules of the calculation engine. Any artifact you generate can import them from /engine/pipeline.js.
    • /templates/ — starter scaffolds (standalone HTML calculator, Node CLI, …) that import the live engine.

What the corpus contains

  • docs — markdown in /docs/
  • class — Rabbi Zajac's Rambam class transcripts on KH 12-16 (hosted with permission; credit Rabbi Zajac and link to Chabad.org when quoting)
  • step — all 21 Rambam pipeline steps with Hebrew names, chapter refs, formulas, and teaching notes
  • galgal — 9 celestial spheres (sun and moon sub-galgalim with Rabbi Losh's pedagogical framing)
  • chapter — KH 11-19 with Sefaria links
  • concept — sun galgalim, moon galgalim, emtzoi vs. amiti, nekudah hanichaches
  • source_type — provenance legend (Rambam / Interpolated / Deduced / Losh)
